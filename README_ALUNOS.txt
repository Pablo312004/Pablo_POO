Projeto: Sistema de Gestão de Notas dos AlunosDescriçãoEste projeto consiste em um sistema simples de gestão de notas dos alunos, que lê dados de um arquivo CSV, processa esses dados para calcular a média das notas, identifica a maior e menor nota, e exibe essas informações em uma interface gráfica usando a biblioteca Flet.Funcionalidades1. Criação do arquivo CSV com dados dos alunos.2. Leitura dos dados do arquivo CSV.3. Cálculo da média das notas.4. Escrita dos dados processados em um novo arquivo CSV.5. Identificação da maior e menor nota.6. Exibição dos dados em uma interface gráfica interativa.Estrutura do Projeto- students.csv: Arquivo CSV com os dados iniciais dos alunos.- students_with_average.csv: Arquivo CSV gerado com os dados dos alunos incluindo a média das notas.- main.py: Script Python principal contendo todas as funcionalidades do projeto.- README.txt: Arquivo de documentação do projeto.Pré-requisitos- Python 3.7 ou superior.- Biblioteca Flet.Instalação1. Clone este repositório para o seu ambiente local:    git clone https://github.com/seu-usuario/sistema-gestao-notas.git    cd sistema-gestao-notas2. Crie e ative um ambiente virtual (opcional, mas recomendado):    python -m venv venv    source venv/bin/activate # Para Linux/Mac    venv\Scripts\activate    # Para Windows3. Instale as dependências do projeto:    pip install fletUso:1. Execute o script principal:    python main.py2. A aplicação abrirá uma interface gráfica exibindo:    - A média das notas dos alunos.    - A maior e menor nota, juntamente com o nome do aluno correspondente.    - Uma tabela com os dados dos alunos (nome, idade e nota).ContribuiçãoSe você deseja contribuir para este projeto, siga os passos abaixo:1.
